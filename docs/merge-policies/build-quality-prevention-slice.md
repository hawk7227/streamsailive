# build-quality-prevention-slice

## Allowed
- `scripts/**`
- `.github/workflows/**`
- `docs/merge-policies/**`
- `package.json` (scripts only)
- `.github/pull_request_template.md`
- `docs/streams-current-status.md` (status updates only)

## Forbidden
- product runtime files
- provider routes
- DB migrations
- unrelated UI feature files
